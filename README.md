Productivity Suite - Single-file Python Project

Files:
- productivity_suite.py : main application (Tkinter)
- notes/ directory will be created automatically next to the script on first run.

How to run:
1. Download the zip and extract.
2. Run: python productivity_suite.py
(Requires Python 3.x; no external packages.)

Notes:
- Notes are saved in ./notes as .txt files.
- File Organizer moves files into subfolders inside the chosen folder by extension.
- Timer supports countdown and stopwatch.
